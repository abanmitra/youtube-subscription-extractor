## How to Add Topics on GitHub

### Step-by-Step Instructions

1. **Navigate to your repository** on GitHub
2. **Click the gear icon** (‚öôÔ∏è) next to "About" in the top-right
3. In the modal that opens, find the **"Topics"** field
4. **Type each topic** and press Enter after each one
5. The field will show up to **30 topics** but recommend **15-20**
6. Click **"Save changes"**
